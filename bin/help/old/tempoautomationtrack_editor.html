Tempo automation track

Time-varying tempochanges (accelerando/ritardando) can be defined using break point lines in the special tempo automation track.

Tips:

  • Press Left Alt + Return to add a node with the value 1.
  • Press Backspace to delete a node.
  • Press Left Alt + X to delete all nodes.
  • Press Left Alt + C to copy all nodes.
  • Press Left Alt + V to paste nodes.
  • To adjust accelerando and deaccelerando behavior, go to File -> Song properties.
