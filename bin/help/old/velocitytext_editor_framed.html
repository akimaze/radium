Velocity text

Velocity text is an alternative way to edit the velocities of a note. Editing text is often quicker than editing velocities graphically.

To enable Velocity text, right-click inside a track and enable "Velocity text". 

Velocity text format: xxt, where
      
xx  = Velocity value. 0 = lowest velocity, ff = highest velocity. (hex format)
  t = Whether to glide to the next velocity or not (denoted by '|').

Tips:

  • Press 'G' to quickly add an ff node.
  • Press 'T' to switch glide mode (i.e. add or remove the | character).
  • Press 'Left Alt + Y' to turn on/off velocity text for the current track.
  • Press 'Left Ctrl + Y' to turn on/off velocity text for all tracks in block.
