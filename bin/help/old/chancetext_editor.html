Chance text

Chance text lets you set the probability for this note or pitch to play.

To enable Chance text, right-click inside a track and enable "Chance text". 

Chance text format: xx, where 

 0 = Only play if the previous note played. For instance if you want to play a chord with same chance.
 1 = 0.004% chance.
..
40 = 25% chance.
.. 
80 = 50% chance.
..
c0 = 75% chance.
..
ff = 99.6% chance.
   = 100% chance. (empty)

  • To remove a chance text (to make sure the note is always played), press DEL.
  • To control the chance of individual voices in the note duplicator, you can automate the instrument effects System Chance 1, System Chance 2, and so forth, instead. (Note that the value ff is interpreted as 256 when automating the chance of individual voices in the note duplicator so you don't have to worry about the voices not always playing.)
