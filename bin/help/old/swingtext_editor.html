Swing

The Swing tracks provide efficient ways to create swing.

There are two types of swing tracks in the editor:

  1. The global swing track.
    • Position: To the right of the "LZ" track. (See first picture.)
    • Task: Apply swing to all tracks in a block.
  2. Swing subtracks.
    • Position: To the left of note text. (See second picture.)
    • Task: Apply swing to individual tracks, overriding the global swing track for that track.
    • How to enable: Right-click inside a track and enable "Swing text". 

Swing text entry format: xxt, where
    
xx  = Swing weight. 1 = lowest weight, 99 = highest weight.
  t = Whether to gradually change tempo between the current and the next swing weight.

If t is " " (empty), the change happens gradually.
If t is "|", the change happens immediately.

Press T to switch between the two modes.

Tips:

  • Press DEL to remove a swing entry.
  • To stop swing in the middle of a block, add "1" (or any number) to the first beat in a bar, and remove all other swing values from that bar.
  • To disable all swing in a block (can be used to quickly compare swing vs. non-swing), unselect the check box below "Swing" in the upper left part of the editor.
  • To adjust accelerando and deaccelerando behavior (relevant when t is empty), go to File -> Song properties.
  • To change whether to let the editor graphics swing along while playing, go to File -> Song properties.
  • To change whether to apply global swing to the tempo sent to plugins and JACK transport, go to File -> Song properties.
