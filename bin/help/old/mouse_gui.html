Mouse

  • Finetuning:
    • Press Ctrl.
  • Y direction only:
    • Press Shift.
  • X direction only:
    • Press a Windows / Cmd key.
  • Move all automation nodes simultaneously:
    • Press Alt.
  • Reset automation:
    • Press Shift + Alt, and click Right mouse button.
  • Break grid:
    • Press Ctrl. (i.e. do finetuning)
  • Delete object under mouse:
    • Press Shift and click Right mouse button.
      This works for automation node, sound object, plugin, track, block, note, etc.
  • Select/unselect something:
    • Press Ctrl key and click Left mouse button.
  • Zoom in or out:
    • Press Ctrl key and use the Mouse scroll wheel.
  • Scroll vertically:
    • Use the Mouse scroll wheel.
    • Below the timeline area in the sequencer you also need to hold a Windows / Cmd key.
    • Note that the editor works differently. In the editor, the mouse scroll wheel does the same as Arrow Down and Arrow Up.
  • Scroll horizontally:
    • Press Shift and use the Mouse scroll wheel.
    • Note that the editor works differently. In the editor, the mouse scroll wheel does the same as Arrow Down and Arrow Up, and those keys will by default zoom lines in/out, or move the cursor 8 steps up/down, depending on which Shift key is pressed.
  • Play song
    • Use the Mouse scroll wheel inside the sequencer or in the bottom bar timelines. Move the mouse cursor and Scroll up to start playing / Scroll down to stop playing. (Note that this behavior can be configured under Edit -> Preferences -> Sequencer)
